# Тестовое задание Java. Startext.
### Стэк технологий:
* Java SE 17
* Spring Boot
* Spring Data JPA
* Spring Web
* Spring Security
* h2
* JUnit
* Apache Maven

* java -jar target/stx-test-task-rest-0.0.1-SNAPSHOT.war
